# ПрофСтарт — MVP веб-платформы виртуальных практик

## 🚀 Быстрый старт

```bash

cd profstart

# 2. Установите зависимости (один раз)
npm install --legacy-peer-deps

# 3. Запустите платформу
npm start
```

Откроется браузер на `http://localhost:3000`

## 🎯 Что реализовано

### Авторизация и роли
- Три роли: студент, преподаватель, администратор
- Вход/регистрация, хранение в localStorage
- Адаптивный интерфейс по роли

### Студент
- Список назначенных практик
- **Виртуальный цех (3D)** — Three.js сцена:
  - Токарный станок с патроном, суппортом, панелью управления
  - 7 шагов технологической карты с проверкой последовательности
  - Подсветка объектов (зелёная рамка) в обучающем режиме
  - Вращение патрона при запуске, движение суппорта при точении
  - Orbit Controls (вращение, зум)
- **Цифровой двойник предприятия**:
  - 8 параметров с ползунками (сырьё, ФОТ, закупки, смены...)
  - Мгновенный пересчёт себестоимости, прибыли, маржи
  - Скидки поставщика при объёме закупок 900+/1000+ единиц
  - Целевые показатели (снижение с/с на 5%)
  - История попыток с графиком
- Обучающий режим: подсказки, тултипы, блокировка неверного порядка
- Экзаменационный режим: без подсказок, фиксация всех ошибок
- Личный кабинет: история, прогресс, вкладка «Мои ошибки»
- Рейтинг студентов

### Преподаватель
- Создание практик из библиотеки сцен
- Настройка технологической карты и критериев
- Переключение режима (обучение ↔ экзамен) одной кнопкой
- Загрузка CSV-файла с данными предприятия
- Статистика по группе: распределение оценок, частые ошибки, таблица студентов
- Экспорт отчёта в CSV

### Администратор
- Загрузка 3D-моделей (.gltf/.glb)
- Управление библиотекой сцен
- Полное CRUD-управление пользователями
- Сводная аналитика платформы: графики, активность, сводка

## 🏗️ Стек технологий

- **React 18** — UI
- **Three.js + @react-three/fiber + @react-three/drei** — 3D сцена
- **Recharts** — графики и аналитика
- **localStorage** — хранение всех данных (без сервера)
- **PapaParse** — парсинг CSV
- **CSS custom properties** — дизайн-система

## ⚙️ Требования

- Node.js >= 16
- npm >= 8
- Браузер: Chrome, Firefox, Edge (последние версии)
- Компьютер с WebGL (для 3D сцены)

---

## 🔧 Известные ограничения MVP

1. Данные хранятся в `localStorage` браузера (без сервера)
2. 3D сцена — процедурные примитивы (не импортированные .gltf модели)
3. Загрузка .gltf/.glb файлов — имитация (файл читается, но не рендерится)
4. Нет онлайн-синхронизации между пользователями
