# ПрофСтарт — Виртуальная производственная платформа

Платформа виртуальных практик для учебных заведений: 3D-тренажёр токарного станка и цифровой двойник предприятия.

## 🚀 Быстрый старт (локально)

```bash
npm install
npm start
```

## 📋 Что улучшено

### 1. 3D-сцена токарного станка
- Добавлены новые детали: шпоночный паз ходового вала, балансировочные отверстия патрона, зубцы кулачков, подшипниковые опоры ходового винта
- Добавлен индикатор часового типа на магнитной стойке
- Расширена мастерская: пожарный щит, верстак с тисками, стеллаж с инструментом, плакат ТБ
- Улучшены стружка и брызги СОЖ
- 3D-подписи деталей при подсветке

### 2. Цифровой двойник — 5 заданий
- **Задание 1**: Снижение себестоимости (−8%)
- **Задание 2**: Оптимизация смен (прибыль > 3.5 млн ₽)
- **Задание 3**: Ценообразование (маржа > 30%)
- **Задание 4**: Инвестиционный анализ (ROI ≥ 20%)
- **Задание 5**: Антикризисный план (кризис поставок ≤ 600 ед.)
- Улучшенные ползунки с анимированным треком и всплывающими подсказками
- Кольцевой прогресс-индикатор, структура затрат, история расчётов
